**CS @ King's College London** · **MSc Data Science & ML @ UCL (incoming)** · Building at the intersection of quantitative finance and machine learning

I build end-to-end systems — from research and backtesting through to live execution and monitoring. Currently focused on algorithmic trading infrastructure and applied ML.

### What I'm building

| Project | Description |
|---|---|
| [**AlphaLab**](https://github.com/bernardoguterres/AlphaLab) | Desktop backtesting platform — 50+ indicators, Monte Carlo simulation, walk-forward validation. React/TypeScript + Flask + Tauri (Rust) |
| [**AlphaLive**](https://github.com/bernardoguterres/AlphaLive) | 24/7 live execution engine deployed on Railway. Consumes AlphaLab strategies, trades via Alpaca, Telegram alerts |
| [**AlphaSignal**](https://github.com/bernardoguterres/AlphaSignal) | Financial RAG system — ingests SEC EDGAR filings & news, hybrid BM25 + dense retrieval, cross-encoder reranking |
| [**nbaMVPPredictor**](https://github.com/bernardoguterres/nbaMVPPredictor) | ML ranking model (Ridge/RF/XGBoost) across 33 seasons, 52 engineered features, SHAP analysis, time-aware backtest |
